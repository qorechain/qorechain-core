# AI Sidecar

The AI Sidecar is a gRPC service that provides AI-powered transaction analysis,
contract auditing, fraud detection, and network optimization for QoreChain validators.

## Features

- Transaction anomaly detection and risk scoring
- Smart contract security auditing
- Fraud pattern analysis
- Fee prediction and optimization
- Network health advisory

## Build

The sidecar binary is distributed as a pre-built Docker image for licensed validators.
See the [QoreChain documentation](https://qorechain.io/docs) for deployment instructions.

## Proto Definition

The gRPC service interface is defined in `proto/ai_sidecar/v1/service.proto`.
